## Run Locally

Clone the pacman-xidian

```bash
  git clone https://github.com/Zamidmustefa/pacman-xidian.git
```

Go to the project directory

```bash
  cd pacman-xidian
```

Install dependencies

```bash
  pip install pygame
```

Start the server

```bash
  python pacman.py
```
